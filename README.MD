## Beauty Procedures

# How to use

1. Install dependencies

    ```bash 
    npm install
    ```

2. Start server both in back and front

    ```bash
    cd back
    npm start

    cd front
    npm run dev
    ```

3. Open front in browser

    ```bash
    http://localhost:5173
    ```

# You can update user role through Postman API

http://localhost:3000/api/v1/users/"USER_ID" use PATCH method
